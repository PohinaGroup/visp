# Julkaisut ja päivitykset

VISP koostuu portaalista, API:sta, mediarelaysta, mobiilisovelluksesta ja OBS-lisäosasta. Käytä tuotannossa saman julkaisun yhteensopivia versioita ja lue aina version julkaisutiedot.

Operaattorin päivitysjärjestys on: varmuuskopio, palvelukuvat, tietokantamigraatiot, web/API, mediarelay ja lopuksi asiakasohjelmat. Tee savutesti kirjautumiselle, julkaisemiselle, OBS-kuvalle, tunnusten kierrätykselle ja etäohjaukselle.

Palauta edellinen versio vain, jos sen tietokantarakenne on yhteensopiva. Muussa tapauksessa palauta myös ennen päivitystä otettu varmuuskopio.
