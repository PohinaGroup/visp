# VISP

VISP välittää puhelimen tai selaimen kameran turvallisesti kotona toimivaan OBS Studioon. Kohtaukset, grafiikat, hälytykset ja lähetysavain pysyvät omassa tuotannossasi.

## Valitse seuraava vaihe

- [Aloita tästä](/fi/docs/get-started), jos käytät VISPin ylläpitämää palvelua.
- [Puhelin- ja selainsovellus](/fi/docs/phone-app) kertoo kameran julkaisemisesta.
- [OBS-etäohjaus](/fi/docs/obs-remote-control) yhdistää VISP-lisäosan.
- [Itse ylläpito](/fi/docs/self-hosting) on palvelimen ylläpitäjille.

VISP ei tarvitse Twitchin, Kickin tai muun alustan lähetysavainta. OBS lähettää lopullisen ohjelman suoratoistopalveluun tavalliseen tapaan.
